# 🍽️ Recipe Search App

A responsive web application that allows users to search for recipes by ingredient using **TheMealDB API**. The application fetches recipe data dynamically, displays recipe cards with images and ingredient previews, and provides direct links to complete cooking instructions.

---

## 📖 Overview

The Recipe Search App is built using **HTML**, **CSS**, and **Vanilla JavaScript**. Users can enter an ingredient, and the application retrieves matching recipes from TheMealDB API. Each result displays the recipe image, title, a short ingredient preview, and a link to view the full recipe.

---

## ✨ Features

- 🔍 Search recipes by ingredient
- 🌐 Fetch recipe data from TheMealDB API
- ⚡ Asynchronous API requests using `fetch()` and `async/await`
- 📋 Dynamic recipe card generation
- 🥘 Displays recipe images and ingredient previews
- 🔗 Direct links to full recipe instructions
- ⌨️ Supports searching with the **Enter** key
- ❌ User-friendly error handling for invalid searches and network issues
- 📱 Responsive layout for desktop and mobile devices

## 💡 How It Works

1. The user enters an ingredient.
2. The application sends a request to TheMealDB API.
3. Matching recipes are retrieved.
4. Detailed recipe information is fetched for each meal.
5. Recipe cards are generated dynamically and displayed on the page.
